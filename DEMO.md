## ⏱️ 빠른 데모 (3분)

### Step 1 — 랜딩 페이지 (30초)
**URL**: https://ai-court.pages.dev/

확인 포인트:
- 프리미엄 다크 테마 + 골드 그래디언트로 "법정" 분위기 연출
- 3가지 서비스 카드 (추천 배지, 무료 배지, Beta 배지)
- 우측 상단 다크/라이트 모드 토글 → 클릭해서 전환 시연
- 하단 통계 카드: GPT-4o / 7단계 / 3인 / 8개+

---

### Step 2 — 가상 재판 시뮬레이션 (90초)
**"가상 재판 시뮬레이션" 카드 클릭**

#### 입력값 (복붙 사용)
```
원고 측:
임대인(집주인)이 계약 만료 후 2개월이 지났음에도 보증금 3,000만 원을 반환하지 않고 있습니다. 
이사까지 완료했으며 집 상태도 양호합니다. 입금 요청을 여러 차례 했으나 무시당하고 있습니다.

피고 측:
세입자가 퇴거 전 집 내부에 심각한 파손(도배 전체 교체 필요, 싱크대 파손)을 남겼습니다.
수리비 견적이 약 500만 원으로, 이를 보증금에서 공제해야 하므로 전액 반환이 불가합니다.
```

확인 포인트:
- **CourtRoom UI**: 판사·검사·변호사 3인 좌석 + 발언자 골드 Glow 애니메이션
- **자동 재생(Auto-play)**: 라운드가 자동으로 진행됨
- **스트리밍**: 각 AI 발언이 실시간으로 타이핑되듯 생성
- 7라운드 후 **판결 분석 카드** 자동 등장:
  - ⭕ 승소 확률 원형 게이지
  - 📋 핵심 쟁점 3가지
  - ✅ 유리한 포인트 / ❌ 불리한 포인트
  - 📌 권고사항

- 하단 **"오늘 남은 횟수: 3/3"** 표시

## 🎯 심사위원 질의응답 예상

### Q: "실제 법률 조언과 어떻게 다른가요?"
> AI Court는 법률 **정보 시뮬레이션** 도구입니다. 실제 법원 판결을 예측하는 것이 아니라, 사용자가 법률 전문가와 상담하기 전에 상황을 정리하고 전략을 세우는 것을 돕습니다. 면책 조항을 명확히 고지합니다.

### Q: "왜 Cloudflare Pages인가요?"
> API 키 보안이 핵심입니다. Cloudflare Pages Functions가 브라우저와 OpenAI 사이의 프록시 역할을 하여 클라이언트에 API 키가 노출되지 않습니다.

### Q: "GPT-4o 비용이 문제되지 않나요?"
> Free 티어에 일일 사용량 제한(상담 3회, 재판 1회, 문서 2회)을 구현했습니다. 재판 시뮬레이션 1회당 약 $0.05-0.10 수준이며, Pro 구독 모델(₩9,900/월)로 지속 가능한 수익 구조를 확보합니다.

### Q: "실제 법률 데이터베이스를 사용하나요?"
> 현재는 GPT-4o의 학습 데이터(한국 법령, 판례 포함)를 활용합니다. 향후 국가법령정보공단 API, 판례 DB와 연동하여 정확도를 높이는 것이 다음 단계입니다.

### Q: "경쟁 서비스와의 차별점은?"
> 기존 법률 AI 챗봇(로톡 AI 등)은 Q&A 형식입니다. AI Court는 **판사·검사·변호사 3인 역할극**을 통해 양측 입장을 동시에 분석하는 **법정 시뮬레이션** 방식이 핵심 차별점입니다. 사용자가 재판관이 되어 전체 공방을 지켜보는 몰입형 경험을 제공합니다.
